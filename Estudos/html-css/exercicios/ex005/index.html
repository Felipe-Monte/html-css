Digitação com semântica

Negrito

Em negrito podem ser escrito dessa forma para caracterizar (não semantica)

Ou da maneira mais semantica usando a tag STRONG dessa maneira

Marcações de texto

Podem ser feitos utilizando a tag MARK dessa maneira

Textos tachados

Podem ser escrito dessa forma com a tag DEL

Textos em italico

O ideal para ulizar o itálico, dessa maneira seria utilizar a tag EM (forma semantica)

Para estilo apenas a tag I dessa maneira

Sublinhado

De forma semantica com a tag INS dessa maneira

Textos subscrito e subscritos

Dessa maneira para palavras como x20 + 3 com a tag SUP

Ou para palavras como H2O com a tag SUB

Uso do lorem (Encher bla bla bla) tag Lorem minuscula
